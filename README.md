# Portfolio Projects – Web Development

## Project 1: SparkleClean – Cleaning Service Website
**Description:**  
SparkleClean is a fictional website for a cleaning service, completely designed and implemented by me. The goal was to create a modern, attractive, and user-friendly landing page that engages visitors and clearly presents information.

**Technologies:**  
- React  
- Tailwindcss  
- Supabase  
- Responsive Design  

**Highlights & Challenges:**  
- Modern, **well-structured layout** for both desktop and mobile  
- Interactive elements such as buttons and scroll effects  
- Optimized for **responsiveness and fast loading times**
- Reserve an Appointment.
